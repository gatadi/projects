To create a new app
> rails new demo_app

To start the rails server (rails comes with WEBrick webserver)
> rails server

Rails console
> rails console
> exit

To generate CRUD with scaffolding
> rails generate scaffold User name:string email:string
> rails generate scaffold Micropost content:string user_id:integer

To create the db
> rake dm:create

To migrate the db
> rake db:migrate VERSION=0
> rake db:rollback

To configure Rails to use RSpec in place of Test::Unit
> rails generate rspec:install

To create controller
> rails generate controller FooBar 

To create a model
> rail generate model User first_name:string last_name:string phone:integer

To destroy/undo-things
> rails destroy model User
> rails destroy controller FooBar

> bundle install
> bundle exec rspec
> bundle exec spork --bootstrap
